# 取件码助手（CodeHelper）

一款为老人场景设计的本地短信取件码助手。

## 功能

- 首屏默认读取最近 12 小时短信，只展示未取件码
- 支持 `12 小时`、`1 天内`、`2 天内`、`5 天内` 快速筛选，切换后立即重新读取短信
- `未取件码` 统计永远只计算未取件数量
- 点击 `强制刷新所有` 会重新加载当前时间范围内的全部取件码，同时展示未取件与已取件
- 未取件卡片支持左滑抽屉式展开 `已取件` 按钮，点击后立即标记为已取件并从默认列表消失
- 已取件卡片在 `强制刷新所有` 结果中始终排在未取件之后；未取件按短信时间倒序排列
- 卡片展示取件状态、接收时间、取件码、短信内容、命中规则和 `双击打开短信` 提示
- 支持设置页维护自定义正则规则，保存后即时刷新列表
- 重新设计了更符合“短信取件码助手”语义的应用图标
- 全程离线运行，不申请网络权限

## 构建

```bash
./gradlew assembleRelease
```

推送到 `main` 后，GitHub Actions 会执行构建校验并上传 APK artifact。
推送 `v*` 标签后，GitHub Actions 会在 Releases 页面创建正式版本并附带可下载 APK。
