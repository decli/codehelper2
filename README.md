# 取件码助手（CodeHelper）

一款为老人场景设计的本地短信取件码助手。

## 功能

- 首屏默认读取最近 12 小时短信，只展示未取件码
- 首页按原型图拆分为 `时间筛选`、`强制展示所有 / 设置`、`未取件统计` 三块区域，适合老人直接查看
- 时间筛选默认选中 `12小时`，点击 `选择其它时间` 可下拉选择 `1天内`、`2天内`、`3天内`、`5天内`、`7天内`、`14天内`，选择后立即重新读取短信
- `未取件码` 统计永远只计算未取件数量
- 点击 `强制展示所有` 会重新加载当前时间范围内的全部取件码，同时展示未取件与已取件
- 未取件卡片支持左滑抽屉式展开 `已取件` 按钮，点击后立即标记为已取件并从默认列表消失
- 已取件卡片在 `强制展示所有` 结果中始终排在未取件之后；未取件按短信时间倒序排列
- 卡片展示取件状态、接收时间、大号居中的取件码、短信内容、`命中规则1/2/...` 标签和 `双击打开短信` 提示
- 设置页支持新增、删除、保存正则规则，内置 `恢复默认设置`，规则存在空白或语法错误时不允许保存
- 重新设计了更符合“短信取件码助手”语义的应用图标
- 全程离线运行，不申请网络权限

## 构建

```bash
./gradlew assembleRelease
```

推送到 `main` 后，GitHub Actions 会执行构建校验并上传 APK artifact。
推送 `v*` 标签后，GitHub Actions 会在 Releases 页面创建正式版本并附带可下载 APK。
