# 取件码助手 (CodeHelper)

专为老人设计的本地短信取件码提取工具。自动读取手机短信，提取快递取件码并以大字体卡片展示，方便老人快速查看和确认。

## 功能特性

- **自动提取取件码**：读取手机短信，根据正则规则自动识别并提取取件码（如 `62667148`、`17-3-18014`、`Y0986`）
- **老人友好界面**：大字体、高对比度、简洁布局，专为老人场景优化
- **时间筛选**：支持 12 小时、1 天、2 天、5 天快速筛选，默认展示 12 小时内未取件码
- **已取件/未取件管理**：左滑卡片标记「已取件」，点击后自动隐藏；「强制刷新所有」可回看全部记录
- **双击跳转短信**：双击取件码卡片可跳转到系统短信应用查看原始短信
- **自定义规则**：支持自定义正则表达式提取规则，灵活适配各类短信格式
- **完全离线**：应用不联网，所有数据在本机处理，保护隐私安全

## 界面设计

- 首屏展示取件码列表，卡片包含：取件状态、时间、取件码（大字体）、短信内容、命中规则
- 顶部区域：应用标题、未取件码统计、时间筛选、操作按钮
- 未取件码排在前面（按时间倒序），已取件码排在后面
- 应用图标：快递包裹 + 取件码标签，一目了然

## 技术架构

- **语言**：Kotlin
- **UI 框架**：Jetpack Compose + Material 3
- **架构模式**：MVVM（ViewModel + StateFlow）
- **数据持久化**：DataStore Preferences
- **最低版本**：Android 8.0 (API 26)
- **目标版本**：Android 16 (API 36)

## 构建方式

```bash
./gradlew assembleRelease
```

APK 输出路径：`app/build/outputs/apk/release/`

## CI/CD

推送代码到 `main` 或 `claude/*` 分支后，GitHub Actions 自动编译并发布 APK 到 Release 页面。

## 权限说明

- `READ_SMS`：读取手机短信以提取取件码（仅本地处理，不联网）
